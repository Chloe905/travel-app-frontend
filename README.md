# Travel App Frontend

這是旅行導覽應用程式的前端專案，使用 React + Vite 建立，具備快速開發與模組化架構，適合進行多語言旅遊資訊展示與即時互動。

---

## 📦 使用技術與版本

| 技術 | 版本 |
|------|------|
| React | ^19.1.0 |
| React DOM | ^19.1.0 |
| Vite | ^5.2.10 |
| Node.js | v18.x |
| @vitejs/plugin-react | ^4.2.1 |
| ESLint | ^9.30.1 |

---

## 🔧 開發環境需求

- Node.js 版本 **18.x**（建議搭配 `nvm` 管理）
- npm 版本 **9 或以上**
- 建議使用 Visual Studio Code 編輯器
